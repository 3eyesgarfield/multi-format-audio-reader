# Offline dictionary

Drop **`cedict_ts.u8`** (CC-CEDICT, 中→英) into this folder to enable offline
word lookup in the reading view (select a word → dictionary card).

Download CC-CEDICT:
- https://www.mdbg.net/chinese/dictionary?page=cc-cedict
- Extract `cedict_ts.u8` here.

Without this file the word card still works but shows "未安装离线词典"; you can
still add words to the vocabulary book and read selections aloud.
